# 🤰 Бот порад для вагітних

## Структура файлів helloworld

```
pregnancy_bot/
├── bot.py           # Головний файл бота
├── database.py      # База даних (SQLite)
├── tips.py          # Завантаження порад з Excel
├── requirements.txt # Залежності Python
├── railway.toml     # Налаштування деплою
└── tips.xlsx        # ТВІЙ файл з порадами (додаєш сама)
```

---

## 📊 Формат таблиці tips.xlsx

Перший рядок — заголовки (обов'язково):

| day | title | text |
|-----|-------|------|
| 1   | Вітаємо! | Перший день вагітності — особливий... |
| 7   | Тиждень минув | Твоє тіло вже змінюється... |
| 14  | | Порада без заголовка — теж ок |

- **day** — день вагітності (від 1 до 280)
- **title** — заголовок (можна залишити пустим)  
- **text** — текст поради (обов'язково)

Один рядок = одна порада. На один день може бути кілька порад.

## 🚀 Покрокова інструкція деплою

### Крок 1. Отримати токен бота

1. Відкрий Telegram, знайди **@BotFather**
2. Напиши `/newbot`
3. Придумай назву (наприклад: `Поради для вагітних`)
4. Придумай username (наприклад: `pregnancy_tips_bot`)
5. BotFather надішле тобі **токен** — збережи його!

### Крок 2. Підготувати файли

1. Завантаж всі файли з цієї папки
2. Заповни `tips.xlsx` своїми порадами
3. Переконайся, що перший рядок містить заголовки: `day`, `title`, `text`

### Крок 3. Завантажити на GitHub

1. Зареєструйся на [github.com](https://github.com)
2. Натисни `+` → `New repository`
3. Назви репозиторій `pregnancy-bot`, натисни `Create`
4. Завантаж всі файли через кнопку `uploading an existing file`
5. Натисни `Commit changes`

### Крок 4. Деплой на Railway

1. Зареєструйся на [railway.app](https://railway.app) через GitHub
2. Натисни `New Project` → `Deploy from GitHub repo`
3. Вибери свій репозиторій `pregnancy-bot`
4. Railway автоматично запустить бота

### Крок 5. Додати токен бота

1. У Railway відкрий свій проект
2. Перейди у вкладку **Variables**
3. Натисни `Add Variable`:
   - Name: `BOT_TOKEN`
   - Value: (встав свій токен від BotFather)
4. Натисни `Add`
5. Railway автоматично перезапустить бота

### Крок 6. Перевірка

1. Знайди свого бота в Telegram
2. Напиши `/start`
3. Вкажи тиждень вагітності
4. Напиши `/today` — має прийти порада!

## ⚙️ Команди бота

| Команда | Опис |
|---------|------|
| `/start` | Реєстрація (вказати тиждень) |
| `/today` | Отримати сьогоднішню пораду |
| `/restart` | Змінити термін вагітності |

---

## 🔄 Як оновити поради

1. Відредагуй `tips.xlsx` локально
2. Завантаж оновлений файл на GitHub (перетягни у репозиторій)
3. Railway автоматично перезапустить бота з новими порадами

## ❓ Часті питання

**Коли приходять поради?**  
Щодня о 9:00 за київським часом.

**Що якщо на якийсь день немає поради?**  
Бот пропускає цей день — нічого не надсилає.

**Чи можна надіслати кілька порад за один день?**  
Так! Додай кілька рядків з однаковим `day` в Excel.

**Скільки коштує?**  
Railway має безкоштовний план для невеликих ботів ($5 кредитів на старт).
